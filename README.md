"# User Admin Expense Tracker (MERN)"

"## Features"
"- JWT authentication with user/admin roles"
"- Expense CRUD with filters, search, pagination"
"- Weekly and monthly summaries"
"- Admin analytics and user management"
"- React + Tailwind UI with charts"

"## Backend Setup (Node/Express)"
"1. Copy .env.example to .env and fill values."
"2. Install dependencies: npm install"
"3. Run dev server: npm run dev"

"## Frontend Setup (React)"
"1. cd client"
"2. Copy client/.env.example to client/.env"
"3. Install dependencies: npm install"
"4. Start dev server: npm run dev"

"## Admin Access"
"- Set ADMIN_SECRET in .env"
"- During registration, use the Admin invite code field to create an admin"
"- Or promote a user manually in MongoDB by setting role to admin"

"## API Overview"
"- POST /api/auth/register"
"- POST /api/auth/login"
"- GET /api/auth/me"
"- GET /api/expenses"
"- POST /api/expenses"
"- PATCH /api/expenses/:id"
"- DELETE /api/expenses/:id"
"- GET /api/expenses/summary?range=weekly|monthly"
"- GET /api/admin/users"
"- PATCH /api/admin/users/:id/block"
"- DELETE /api/admin/users/:id"
"- GET /api/admin/expenses"
"- GET /api/admin/analytics"
